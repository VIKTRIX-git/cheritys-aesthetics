# 🧶 Cherity's Aesthetics

> Custom handmade knitwear & crochet — made to order.

A professional landing page for **Cherity's Aesthetics**, a custom handmade knitwear and crochet business. Customers describe their design, and we handcraft it exactly to their style, size, and colour.

## 🌐 Live Site

👉 [cheritys-aesthetics.github.io/cheritys-aesthetics](https://VIKTRIX-git.github.io/cheritys-aesthetics)

## 📄 Pages / Sections

- **Home** — Hero with animated thread lines and yarn ball icon
- **About** — Business story and 3-step order process
- **Process** — Visual 4-step banner (Contact → Quote → Create → Receive)
- **Colours** — Spinning colour wheel + colour grid (any colour available)
- **Contact** — WhatsApp, Gmail, Instagram, TikTok, Facebook

## ✏️ How to Edit Contact Info

Open `index.html` and search (Ctrl+F) for these placeholders:

| Placeholder | Replace with |
|---|---|
| `YOUR_WHATSAPP_NUMBER` | e.g. `2348012345678` (no + or spaces) |
| `youremail@gmail.com` | Her actual Gmail address |
| `YOUR_HANDLE` (Instagram) | Her Instagram username |
| `YOUR_HANDLE` (TikTok) | Her TikTok username |
| `YOUR_PAGE` | Her Facebook page URL |

## 🛠️ Built With
